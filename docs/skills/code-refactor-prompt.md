# 代码重构提示

当任务是改进结构而不改变支持的行为时，使用此技能。

## 防护栏

- 无有意行为改变
- 无隐藏契约扩展
- 优选小的、可审查的步骤
- 可能时在每个结构切片后验证
- 保留现有公共名称，除非重命名是任务的一部分

## 允许的操作

- 拆分过大文件
- 提取连贯的辅助函数或模块
- 合并人为的包装器
- 移除无用的兼容性路径
- 将重复逻辑标准化到一个契约背后

## 执行模式

1. 确认当前基线和受保护行为
2. 定义确切的结构目标
3. 进行最小的安全结构切片
4. 运行聚焦验证
5. 如果行为和结构同时开始变化则停止

## 预期输出

- 重构摘要
- 行为保持不变的证明
- 剩余的结构债务（如有）
