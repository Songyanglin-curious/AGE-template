# 计划索引

使用此目录存放重要的实现计划。

- 从 `00-plan-authoring-and-execution-guide.md` 开始
- 每个计划应专注于一个成果界面
- 按状态归档或替代计划，而非删除历史记录
- 对于中小型项目，首选 `docs/plans/YYYY-MM-DD-topic-plan.md`
- 如果计划数量增多，可以使用数字前缀，如 `NNN-YYYY-MM-DD-topic-plan.md`

每个创建的计划需要在实施前进行计划审计，并在完成前进行结项审计。

推荐的文件名：

- `2026-05-21-user-list-plan.md`
- `2026-05-21-role-permission-alignment-plan.md`
- `012-2026-05-21-checkout-validation-plan.md`
